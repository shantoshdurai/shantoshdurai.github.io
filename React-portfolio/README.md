# React Portfolio

Personal portfolio website built with React, Vite, and Tailwind CSS.

## Tech Stack

- **React** — UI framework
- **Vite** — Build tool with HMR
- **Tailwind CSS** — Utility-first styling
- **shadcn/ui** — Component library

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
